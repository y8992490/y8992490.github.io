Scanner对象

我们可以通过Scanner类来获取用户的输入在Java的包里（java.util.Scanner）

代码里就可以这样用了

Scanner s = new Scanner(System.in);

通过Scanner类的next()与nextLine()方法获取输入字符串 （让用户输入）

在读取前我们一般使用hasNext()可以理解为是否还有下一个与hasNextLine()还是否有下一行判断是否还有输入的数据

  • next()
    1. 一定要读取到有效字符后才可以结束输入
    2. 对输入的有效字符之前遇到的空白，next（）方法会自动将其去掉
    3. 只有输入有效字符后才将其后面输入的空白作为分隔符或者结束符
    4. next（）不能得到带有空格的字符串
package Scanner;

import java.util.Scanner;

public class Dome_01 {
    public static void main(String[] args) {
        //创建一个扫描对象，用于接收键盘数据
        Scanner scanner = new Scanner(System.in); //先告诉电脑需要用Scanner这个方法接收数据
        System.out.println("使用next方式接收：");

        //判断用户有没有输入字符串
        if (scanner.hasNext()){                    //让电脑判断用户有没有输入数据并接收数据
            //使用next方式接收
            String str = scanner.next();
            System.out.println("输出的内容是："+str);//让电脑输出接收到的用户输入的数据
            //这个如果输出Hello wroid   这个只会输出Hello 因为这个是以空格符为终止符
        }
        scanner.close();
        //凡是IO流的类如果不关闭就会一直占用资源，要养成良好的习惯用完就管闭

    }
}
  • nextLine()
    1. 以Enter为结束符nextLine方法返回的是回车前的所有字符
    2. 可以获得空白符
package Scanner;

import java.util.Scanner;

public class Dome_03 {                              //运用nextLine方法
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);   //告诉电脑需要运用Scanner方法
        System.out.println("(运用nextLine)请输入");  //运用Scanner中的nextLine方法

        if (scanner.hasNextLine()){                 //判断是否还有输入
            String str = scanner.nextLine();        //告诉电脑运用NextLine来输出得到的数据
            System.out.println("输出的数据"+str);
            //这个如果输出Hello wroid   这个就能够全部输出Hello Worid
            //这个是以回车符为终止符
        }
        scanner.close();                            //将IO流的类结束
    }
}

Scanner进阶使用

package Scanner;

import java.util.Scanner;

public class Dome_05 {
    public static void main(String[] args) {
        /**
         * 1.可以输入多个数字
         * 2.并求其总和和平均数
         * 3.每输入一个数用回车确认
         * 4.通过输入非数字来结束输入并输出结果；
         */
        Scanner scanner = new Scanner(System.in);

        //求和(这里是先解决求和的需求)
        double sum =0;//
        //计算输入了多少次多少个数字
        int m = 0;//m作为记录输入次数
        //通过循环判断是否还有输入，并在里面对每一次进行求和和统计
        System.out.println("请输入");
        while (scanner.hasNextDouble()){
            double x = scanner.nextDouble();
            m = ++m; //这里不能用m++ 为什么自己先想想
            sum = sum + x;
            System.out.println("输入的次数为"+m+"结果是sum+"+sum);
            System.out.println("如需结束请输入非数字");
        }

        System.out.println(m + "输入和总和为" + sum);
        System.out.println(m + "总和的平均数为" + (sum / m));

        scanner.close();
    }
}

顺序结构

  1. Java的基本结构就是顺序结构，除非特别指出，否则就按照顺序一句一句执行。

  2. 顺序结构是最简单的算法结构

  3. 语句与语句之间，框与框之间是按从上到下的顺序进行的，它是由若干个依次执行的处理步骤

    组成的，它是任何一个算法都离不开的一种基本算法结构

  1. if多选结构
    • 真是情况还可能有ABCD等多个区间多级判断，比如90-100是优 良中 差等，在我们生活中很多时候也不仅仅只有两个，所以我们需要一个多选择结构来处理这类问题
package struch;

import java.util.Scanner;

public class Dome_04 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("请输入");
        int score = scanner.nextInt();
        if (score==100){
            System.out.println("满分");
        }else if (score>=80&& score<100){
            System.out.println("A级");
        }else if (score>=70&& score<80){
            System.out.println("B级");
        }else if (score>=60&&score<70){
            System.out.println("C级");
        }else if score<60&&score>=0){
            System.out.println("D级");
        }else {
            System.out.println("输入不合法");
        }
        scanner.close();
        /**
        if语句至少有1个else语句在所有的else if语句之后
        if语句可以有若干else if语句它们必须在else之前
        在其中一个else if语句检测为true，其他的else if以及else都将跳过执行
        */
    }
}

###循环结构

####while循环

while 是最基本的循环，它的结构为

while(){
    //循环内容
}
  1. 只要布尔表达式为true，循环就会一直执行下去
  2. 我们大对数情况是会让循环停止下来的，我们需要一个让表达式失效的方式来结束循环
  3. 少部分情况需要循环一直执行，比如服务器的请求响应监听等。
  4. 循环条件一直为true就会造成无线循环【死循环】，我们正常的业务编程中应该尽量避免死循环

会影响程序性能或者造成程序卡死崩溃

  1. 思考1+2+3+—–100=？
package Cycle;

public class Dome_01 {//实现从1+到100

    public static void main(String[] args) {
        int a = 2;
        int b = 1;
        while (a<101)
        {
        b = a+b;
        a++;
            System.out.println("结果为："+b);
        }
    }
}

知道一个求和公式，就是（首项+末项）乘以项数除以2，首项就是第一个数，在这里是1，末项就是最后一个数，在这里就是100，项数就是你一共加了多少个数，在这里就是100，所以套在公式里就是：（1+100）×100÷2=5050。

  • do while循环

    1. 对于while而言，如果不满足条件，则不能进入循环。但有时候我们需即使不满足条件

      也至少执行一次

    2. do…..while循环和while循环相似，不同的是，do….while循环至少会执行一次

      do{
    //代码语句
}while(布尔表达式)；

    3. while和do…while的区别

      • while先判断后执行，do…while是先执行后判断
      • Do …..while总是保证循环体会被至少执行一次！这是他们的主要差别。
package Cycle;

public class Dome_02 {
    public static void main(String[] args) {
        //do....while
        int i = 0;
        while (i<0){
            System.out.println(i);//这里可以看见进来后程序判断i不小于0所以不执行后面
            i++;
        }
        System.out.println("//////////////////分割线////////////////");
        do {
            System.out.println(i);
            i++;
        }while (i<0);  //这里可以看见是先进入程序之后判断 所以才会输出下方的结果
    }
}

####for循环

  1. 虽然所有循环结构都可以用while或者do….while表示，但java提供了另一种语句

    for循环，使用一些循环结构变得更加简单。

  2. for循环语句是支持迭代的一种通用结构，是最有效，最灵活的循环结构。

  3. for循环执行的次数实在执行前就确定的，语法格式如下

    for(初始化；布尔表达式；更新){
    //代码语句
}
